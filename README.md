# 🚀 網頁設計競賽 - 個人練習環境啟動指南

本環境專為「選手自主練習」設計，旨在讓你在個人電腦上模擬比賽現場的開發環境。

**注意：** 比賽現場環境通常已由大會預裝完畢，此環境僅供平時練習、開發專案使用。

請注意此專案尚需要在共享雲端上下載 PHP 與 MySQL 主體檔案才為完整，也可直接在雲端中下載完整開發環境。

## 🛠️ 兩步驟啟動練習環境

### 第一步：初始化資料庫 (重買電腦或第一次練習才要跑)

1. 點擊執行 `init_mysql.bat`
2. 看到跳出視窗並顯示 `ready for connection` 後即可關閉
3. 此步驟會在你的資料夾內建立資料庫空間，只需執行一次

### 第二步：進入練習狀態 (每次練習必跑)

1. 點擊執行 `activate.bat`
2. 執行後會自動開啟兩個視窗：
   - **MySQL 視窗：** 這是資料庫後台，請縮小化就好，不要關閉
   - **開發指令視窗：** 標題會顯示 `(Competition_Env)`

## 📋 選手自我檢查 (模擬比賽環境)

在比賽現場或練習環境中，請在「開發指令視窗」輸入以下指令，確保環境已正確就緒。若能噴出正確版本號，代表你可以開始寫程式了：

| 檢查項目 | 輸入指令 | 預期結果 |
|---------|---------|--------|
| PHP 版本 | `php -v` | 顯示 PHP 8.4.6 或對應版本 |
| 套件管理 | `composer -V` | 顯示 Composer version 2.x |
| 資料庫工具 | `mysql --version` | 顯示 mysql Ver 8.4.8 |

## 💡 練習小提醒

### 資料夾移動
這套環境支援隨插即用。你可以把整個資料夾存進隨身碟，換一台電腦後直接執行 `activate.bat` 就能繼續練習。

### 比賽現場差異
比賽時的資料庫可能已經是背景服務，不一定會跳出黑視窗。但在這套練習環境中，請務必保持 MySQL 視窗開啟，你的網頁才能連上資料庫。

### 環境恢復
如果練習到一半資料庫亂掉了，直接刪除 `mysql-8.4.8-winx64/data` 資料夾，重新執行 `init_mysql.bat` 即可獲得全新的資料庫。

### 指令輸入沒反應？
請確認你是在有 (Competition_Env) 字樣的那個視窗輸入指令。
